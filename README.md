## Async Tree

React component for managing an asynchronous tree.

### Features

- Asynchronous children loading.
- Drag-and-drop support for reordering nodes.
- Customizable via props.

### Tech Stack

- **React**
- **Vite**
- **pnpm**

### Getting Started

```bash
cd async-tree
```

```bash
pnpm install
```

```bash
pnpm dev
```
